## How to Run

Open `index.html` in any modern browser. No installation needed.

---

## Features

🍛 **Food Gallery** — 6 traditional Indian dishes in a responsive 3x2 grid  
🎨 **Alternating Card Colors** — Sage green and cream cards for visual contrast  
✨ **Hover Effects** — Cards lift smoothly on hover  
📖 **Detail Pages** — Click any dish to read its story — image and description side by side  
🌈 **Color-Coded Pages** — Each detail page matches its card color  
📱 **Responsive Design** — Works on all screen sizes
